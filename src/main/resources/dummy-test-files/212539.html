Comment 5 Chu Alan "sprite" 2003-07-13 01:34:40 PDT 
brois, 
my understanding to the problem is that when the row of button/checbox/selectbox
is too long to fit in the window, it will push the content of the dialog beyond
the border. In this sense, the problem arises when the font of a particular
system makes that button/checkbox/selectbox bar too long to fit in the window,
no matter it is default or customize font size or not (of course it is a much
more serious problem when the bug is visible when user is using standard/default
font size, but the underlying problem is still the same :))
Comment 6 Boris Zbarsky (:bz) 2003-07-13 01:42:06 PDT 
Yes, I am quite aware of what is causing the problems, thank you.

there are two main ways of fixing bug 171454:

    units and its contents in another).

Depending on which course of action is taken, this bug may or may not get
fixed.

In any case, I think further comments on this bug are superfluous till either
bug 171454 is fixed or this bug is marked duplicate of it.
